NV-UV is a new GeForce RTX 50 undervolting tool built on MSI Afterburner A new utility called NV-UV has surfaced for NVIDIA GeForce RTX 50 series graphics cards. According to PC Games Hardware, the tool was created by community developer cubi2k82 and works as a companion app on top of MSI Afterburner, rather than a replacement for it. It is aimed at Blackwell users who want a simpler way to tune voltage and frequency behavior. The report says NV-UV was made because Blackwell no longer allows direct write access to the VF curve through NVAPI, which makes manual undervolting less convenient than before. Instead of relying on MSI Afterburner’s curve editor alone, NV-UV adds presets and automation on top of the existing backend. That includes one-click Eco, Balanced, Performance, and Max presets, plus more granular manual adjustment options. NV-UV, Source: cubi2k82 One of the more interesting features is a so-called UV-Pilot mode, which can detect supported games and apply matching undervolt profiles automatically. PCGH also says the tool includes an Auto-UV scanner with its own DX12 and DXR stress test, live telemetry, VF curve visualization, idle optimization, and a crash recovery system that can reduce clocks after driver failures. The game database reportedly covers more than 570 titles so far. NV-UV Features at a glance (translation) Community Presets (Eco / Balanced / Performance / Max): Four pre-configured profiles per GPU serve as a starting point. Select a preset, one click, and you’re done. The presets are intentionally conservative. Those who want to get more performance can, of course, adjust the values ​​themselves at any time. Voltage Step Scanner: Automatically finds the highest stable frequency for a defined target voltage. The scanner uses a DX12 + DXR stress test that cycles through various load patterns, including idle-to-full-load transients, DXR ray tracing bursts, and variable load levels. The focus is specifically on the load changes that typically cause instability or driver crashes during gaming. UV Pilot (Automatic Game Detection): Currently automatically detects around 573 games and can assign a separate UV profile to each one. Start game → UV profile is activated. End game → Return to the stock profile or a defined anchor profile. Game Replay (Crash Downstep): If the graphics driver crashes, which can happen with overly aggressive undervolting, NV-UV automatically reduces the frequency. Once you’ve found a stable UV profile, you can close NV-UV; Afterburner will handle the rest, and the profile will remain active. Combining Game Replay with the UV Pilot allows you to use custom, game-specific UV profiles. Mini View: A compact view for the edge of the desktop. All essential functions remain available, except for the VF curve display. This view can optionally remain always in the foreground and be used alongside the original AB Curve Editor. Support is currently listed for GeForce RTX 5090, RTX 5080, RTX 5070 Ti, and RTX 5070 graphics cards . The developer recommends NVIDIA 590-series drivers or newer, Windows 11 64-bit, and an installed copy of MSI Afterburner with voltage control enabled. The tool itself is described as a portable executable with no separate .NET requirement. 1 / 5 ‹ › ⛶ × 1 / 5 ‹ › This is still an early release. NV-UV is currently in closed alpha , and the developer is still asking for more feedback from users. Known issues include scan-related crashes, occasional VF curve display problems, and incomplete validation across all supported GPU models. For now, this looks like an early enthusiast utility for Blackwell owners, not a finished mainstream tuning app. Source: NV-UV (PCGH Forums) via PC Games Hardware Follow us on Google News Set as Preferred Source Share
